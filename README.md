# Pokédex

Application web React permettant de consulter des Pokémon et leurs informations principales.

## Démo

https://pokedex-maj.vercel.app

## Fonctionnalités

- Affichage d’une liste de Pokémon
- Consultation des détails d’un Pokémon
- Affichage des images et informations principales
- Interface responsive

## Technologies utilisées

- React
- JavaScript
- CSS
- HTML

## Installation

Cloner le projet :

```bash
git clone https://github.com/ygherri/pokedex-maj.git
```

Accéder au dossier :

```bash
cd pokedex-maj
```

Installer les dépendances :

```bash
npm install
```

Lancer le projet :

```bash
npm start
```

Ouvrir l’application :

```text
http://localhost:3000/
```

## Build

```bash
npm run build
```
